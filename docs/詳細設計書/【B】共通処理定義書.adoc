== 概要

本書は THE LUXE ROOM（脱出ゲーム）全体で使用する共通ユーティリティの仕様を記載します。
XSS対策・入力バリデーション・UIコンポーネントを対象とします。

---

== 1. XSSProtection クラス

=== 所在

`src/utils/xss.js`

=== 責務

DOM への文字列反映前に特殊文字をエスケープし、XSS 攻撃を防ぐ。

=== 静的メソッド

==== sanitize(value)

文字列内の HTML 特殊文字をエンティティに置換して返す。

[cols="1,3", options="header"]
|===
|引数
|説明

|`value`
|任意の値。非文字列の場合はそのまま返す。
|===

*変換表:*

[cols="1,1", options="header"]
|===
|入力文字
|変換後エンティティ

|`&`
|`&amp;`

|`<`
|`&lt;`

|`>`
|`&gt;`

|`"`
|`&quot;`

|`'`
|`&#x27;`
|===

*実装方針:*
- 変換ルールは `map` オブジェクトで管理し、`replace(/[&<>"']/g, ...)` で一括処理する

== 2. Validator クラス

=== 所在

`src/utils/validator.js`

=== 責務

レベルデータ・視点インデックス・暗証番号入力の形式チェックを行う。

=== 静的メソッド

==== isValidLevel(level)

ゲームレベルオブジェクトの形式を検証する。

[cols="1,1,3", options="header"]
|===
|引数
|型
|説明

|`level`
|`Object`
|ゲームレベルデータ
|===

*判定条件:*

. `level` が truthy であること
. `typeof level === 'object'` であること
. `typeof level.id === 'string'` であること
. `typeof level.code === 'string'` であること

*戻り値:* `true` / `false`

---

==== isValidViewIndex(index, viewsLength)

現在の視点インデックスが有効範囲にあるか検証する。

[cols="1,1,3", options="header"]
|===
|引数
|型
|説明

|`index`
|`number`
|チェック対象のインデックス

|`viewsLength`
|`number`
|視点配列の長さ
|===

*判定条件:*

. `Number.isInteger(index)` であること
. `index >= 0` であること
. `index < viewsLength` であること

*戻り値:* `true` / `false`

---

==== isValidCodeInput(value)

暗証番号入力値の基本形式を検証する。

[cols="1,1,3", options="header"]
|===
|引数
|型
|説明

|`value`
|`string`
|入力値
|===

*判定条件:*

. `typeof value === 'string'` であること
. `value.trim().length > 0` であること（空白のみ不可）

*戻り値:* `true` / `false`

== 4. 使用上の共通ルール

. ユーザー入力値は必ず `XSSProtection.sanitize()` を通してから DOM に反映する
. バリデーションは `Validator` クラスを通じて行い、個別の正規表現を散在させない
. localStorage は本プロジェクトでは使用しない
