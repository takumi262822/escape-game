== 概要

本書はプロジェクト「{project}」に登場するコード値（識別子・種別値）を列挙したものです。
ソースファイル `src/constants/code-definitions.js` から逆引きで抽出しています。

---

== 難易度コード（CodeDefinitions.DIFFICULTY）

ファイル: `src/constants/code-definitions.js`

[cols="2,2,4", options="header"]
|===
|定数名
|コード値
|説明

|DIFFICULTY.EASY
|'EASY'
|初級難易度。謎解きが比較的シンプルな構成

|DIFFICULTY.NORMAL
|'NORMAL'
|中級難易度。複数部屋にまたがる謎を含む

|DIFFICULTY.HARD
|'HARD'
|上級難易度。複雑なアイテム組み合わせと多段謎解き

|===

== アイテムコード（GameConstants.ITEM_HINTS のキー）

ファイル: `src/constants/game-constants.js`

[cols="2,3,4", options="header"]
|===
|アイテムコード
|ヒントテキスト
|用途

|wrench
|古いレンチだ。
|ボルトや機械系オブジェクトに使用

|glasses
|レンズに「2856」と見える。
|ダイヤル錠の数字ヒント取得

|oil
|機械油だ。
|摩擦で開かないオブジェクトに使用

|tape
|絶縁テープ。
|断線オブジェクトへの使用

|bar
|重い鉄の棒。
|重量が必要なオブジェクトに使用

|desk_key
|銀色の小さな鍵。
|キーロックオブジェクトに使用

|===
