== 概要

本書はプロジェクト「{project}」に登場するコード値（識別子・種別値）を列挙したものです。
ソースファイル src/constants/code-definitions.js, src/constants/game-constants.js から逆引きで抽出しています。

---

== 難易度コード（DIFFICULTY）

ファイル: src/constants/code-definitions.js

[cols="1,3,3,2,1,4,3", options="header"]
|===
|項番
|コード和名
|コード英名
|データ型
|桁数
|説明
|記述形式

|1
|初級
|EASY
|文字型
|4
|初級難易度。謎解きが比較的シンプルな構成
|半角英大文字

|2
|中級
|NORMAL
|文字型
|6
|中級難易度。複数部屋にまたがる謎を含む
|半角英大文字

|3
|上級
|HARD
|文字型
|4
|上級難易度。複雑なアイテム組み合わせと多段謎解き
|半角英大文字

|===
