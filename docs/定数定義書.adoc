== 概要

本書はプロジェクト「{project}」で利用する定数値を列挙したものです。
ソースファイル `src/constants/game-constants.js` から逆引きで抽出しています。

---

== 方向定義（VIEWS）

[cols="2,1,2,4", options="header"]
|===
|定数名
|型
|値
|内容

|VIEWS
|string[]
|['NORTH', 'EAST', 'SOUTH', 'WEST']
|部屋内の視点回転順を管理する。move() は VIEWS インデックスを ±1 してループする

|===

---

== タイミング制御（ミリ秒）

[cols="2,1,2,4", options="header"]
|===
|定数名
|型
|値
|内容

|ERROR_MS
|number
|500
|エラー演出（入力不正時のフラッシュ）の表示時間 (ms)

|RELOAD_MS
|number
|5000
|クリア演出後にタイトルへ自動遷移するまでの待機時間 (ms)

|===

---

== クリアメッセージ（CLEAR_MESSAGES）

[cols="2,1,3,4", options="header"]
|===
|定数名
|型
|値
|内容

|CLEAR_MESSAGES.EASY
|string
|【初級踏破】 記憶の断片を回収した。
|初級クリア時に clear-message 要素へ設定するテキスト

|CLEAR_MESSAGES.NORMAL
|string
|【中級完遂】 深淵の迷宮を突破した。
|中級クリア時に clear-message 要素へ設定するテキスト

|CLEAR_MESSAGES.HARD
|string
|【上級超越】 全ての因果を解き明かした。
|上級クリア時に clear-message 要素へ設定するテキスト

|===

---

== アイテムヒント（ITEM_HINTS）

[cols="2,1,3,4", options="header"]
|===

|型
|値
|内容

|ITEM_HINTS.tape
|string

|テープ選択時のテキスト

|ITEM_HINTS.bar
|string
|重い鉄の棒。
|鉄の棒選択時のテキスト

|ITEM_HINTS.desk_key
|string
|銀色の小さな鍵。
|机の鍵選択時のテキスト

|===
